# 🚀 Vorexo Career Portal

## 📌 Project Overview

The **Vorexo Career Portal** is a full-stack web application developed to streamline the recruitment process for employers and job seekers.

This project is **sponsored by Vorexo Company** and focuses on providing a centralized platform where employers can post job openings and candidates can apply efficiently.

---

## 🎯 Objectives

* Simplify job posting and application process
* Provide a user-friendly dashboard for employers
* Enable candidates to apply and track applications
* Build a scalable and modern recruitment platform

---

## ✨ Features

### 👨‍💼 Employer Features

* Secure login system
* Post and manage job listings
* View and manage applications
* Dashboard for tracking job activity

### 👩‍💻 Candidate Features

* Browse available job opportunities
* Apply for jobs with resume upload
* View job details

### ⚙️ System Features

* REST API-based backend
* Secure authentication system
* File upload functionality (resumes)
* Responsive UI design

---

## ⚙️ Installation & Setup

### 🔹 Clone the Repository
git clone https://github.com/omkhandre/Vorexo-Career-Portal.git
cd Vorexo-Career-Portal

### 🔹 Frontend Setup
cd frontend
npm install
npm run dev

### 🔹 Backend Setup
cd backend
npm install
node server.js


## 🔐 Environment Variables
Create a `.env` file in the backend directory and add:

PORT=5000
SUPABASE_URL=your_url
SUPABASE_KEY=your_key
